# Potato_Disease_Detection


AI-driven Potato Leaf Disease Detection System designed to identify and classify diseases in potato leaves using deep learning techniques. Traditional methods of disease detection in agriculture rely heavily on manual inspection, which is time-consuming, labor-intensive, and prone to human error. To address these challenges, the proposed system employs a Convolutional Neural Network (CNN) to classify potato leaf images into three categories: Healthy, Early Blight, and Late Blight.
